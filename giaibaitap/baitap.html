Ví dụ 1: Chứng minh rằng S = 5 + 52 + 53 + 54 + ... + 5100 chia hết cho 6.

Ta có: S = 5 + 52 + 53 + 54 + ... + 5100
= 5( 1 + 5) + 53( 1 + 5) + 55( 1 + 5) + ... + 599( 1 + 5)
= 6( 5 + 53 + 55 + ... + 599 ) ⋮ 6 (Điều phải chứng minh)

Ví dụ 2: Chứng minh rằng S = 1 + 3 + 32 + 33 + 34 + ... + 311 chia hết cho 40.

Ta có: S = 1 + 3 + 32 + 33 + 34 + ... + 311
= ( 1 + 3 + 32 + 33) + 34(1 + 3 + 32 + 33) + 38(1 + 3 + 32 + 33)
= (1 + 3 + 32 + 33)( 1 + 34 + 38)
= 40(1 + 34 + 38) ⋮ 40 (Điều phải chứng minh).

Phương pháp: Vận dụng các tính chất và dấu hiệu chia hết. Phân tích biểu thức thành các tích chứa cơ số chia hết.

Ví dụ 1: Chứng minh tổng 3 số tự nhien liên tiếp chia hết cho 3.

  • Gọi 3 số tự nhiên liên tiếp lần lượt là a, a+1, a+2 với a ∈ N.
  • Tổng 3 số S = a + ( a + 1 ) + ( a + 2 ) = 3a +3 = 3( a + 1 ) ⋮ 3 (Điều phải chứng minh)

Ví dụ 2: Chứng minh tích của 2 số tự nhiên chẵn liên tiếp chia hết cho 8.

  • Gọi 3 số tự nhiên chẵn liên tiếp lần lượt là 2a và 2a + 2 với a ∈ N.
  • Tích 2 số P = 2a . ( 2a + 2 ) = 4a. (a+1)
    • Xét tích a( a + 1) luôn chia hết cho 2 (Vì nếu a là lẻ thì a+1 là chẵn, và ngược lại)
    • Vậy tích 4a. (a+1) luôn chia hết cho 8 (Vì 4.2 = 8) (Điều phải chứng minh)

Ví dụ 1: Tìm số tự nhiên n để 5 chia hết cho n-1

Giải:

Để 5 chia hết cho n-1 ( n ∈ N ) thì n-1 ∈ U(5).

Ta có: U(5) = {1 ,5 }

  • Với n-1 = 1 => n=2
  • Với n-1 = 5 => n=6

Vậy n ∈ { 2, 6} thõa mãn yêu cầu đề bài.

Ví dụ 2: Tìm số tự nhiên n để n+10 chia hết cho n+5

Giải:

Phương pháp : Ta đưa bài toán về dạng cơ bản như ví dụ 1, triệt tiêu vế trái thành số tự nhiên.

Ta có: n+10 ⋮ n+5
=> (n+10) - (n+5) ⋮ (n+5)
=> 5 ⋮ (n+5)
=> n+5 ∈ U(5)

Để 5 chia hết cho n+5 ( n ∈ N ) thì n-1 ∈ U(5).

Ta có: U(5) = {1 ,5}

  • Với n+5 = 1 (Điều này không xảy ra do n ∈ N)
  • Với n+5 = 5 => n=0

Vậy n ∈ {0} thõa mãn yêu cầu đề bài.
